Hướng dẫn chạy web
B0: Tải source code project từ github về
B1: Cài NodeJs(download trên trang chủ về) tại folder project
B2: Mở code bằng Vscode, ấn ctrl + j để mở terminal:
    + Nếu trong folder code không có folder node_module, gõ npm install json-server
    + Nếu đã có: gõ npx json-server db.json
B3: Từ file login.html, open Live Server sau đó đăng nhập tài khoản cilent (trong db.json) 
B4: Tiếp tục trở lại file login.html, nhấn chuột phải để open live server lần nữa, đăng nhập tk admin
