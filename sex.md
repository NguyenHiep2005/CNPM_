Hướng dẫn ae chạy app 

B1: Cài NodeJs(download trên trang chủ về)
B2: Mở code bằng Vscode, ấn ctrl + j để mở terminal:
    + Nếu trong folder code không có folder node_module, gõ npm install json-server
    + Nếu đã có: gõ npx json-server db.json
B3: Nhấn chạy live server để vào trang cilent
B4: Vào link http://localhost:5500/admin.html mở trang admin

B5: Bắt đầu Test (lọ thử phát)
